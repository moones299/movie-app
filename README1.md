 Movie App

یک اپلیکیشن تحت وب برای جستجو، مشاهده جزئیات و امتیاز فیلم‌ها که با React ساخته شده است.  
این پروژه به عنوان نمونه‌کار برای نمایش مهارت‌های من در React، CSSو کار با APIها طراحی شده است.

---

 ویژگی‌ها
-  جستجوی فیلم‌ها بر اساس نام  
-  نمایش پوستر، امتیاز، توضیحات و اطلاعات فیلم  
-  طراحی واکنش‌گرا (Responsive) برای موبایل و دسکتاپ  
-  استفاده از API برای دریافت اطلاعات واقعی فیلم‌ها  
-  رابط کاربری ساده و کاربرپسند  

---

 تکنولوژی‌ها
-  **React.js**  
-  **JavaScript (ES6+)**  
-  **HTML5 / CSS3**  
-  **React Router**  
-  **API Integration**

---

 نحوه اجرا

برای اجرای پروژه در سیستم خودتان، مراحل زیر را انجام دهید:

```bash
# 1️⃣ کلون کردن پروژه از گیت‌هاب
git clone https://github.com/moones299/movie-app.git

# 2️⃣ ورود به پوشه پروژه
cd movie-app

# 3️⃣ نصب وابستگی‌ها
npm install

# 4️⃣ اجرای پروژه
npm start
